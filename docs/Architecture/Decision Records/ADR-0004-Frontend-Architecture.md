# ADR-0004 — Frontend Architecture

## Decision

Adopt Astro as the application framework with React for interactive user interfaces.

## Rationale

- Excellent performance
- Island architecture
- SEO-friendly
- Component interoperability
- Progressive enhancement

## Outcome

Astro is the primary framework.

React is used where client-side interactivity is required.
