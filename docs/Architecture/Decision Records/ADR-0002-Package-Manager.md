# ADR-0002 — Package Manager

## Decision

Adopt **pnpm** as the official package manager.

## Rationale

- Fast installs
- Efficient disk usage
- Excellent workspace support
- Native monorepo capabilities
- Deterministic dependency resolution

## Alternatives

- npm
- Yarn

## Outcome

All packages and applications will use pnpm.
