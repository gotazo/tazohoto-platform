# User Experience Principles

## Principles

- Technology should become invisible.
- Minimize cognitive load.
- Keep workflows intuitive.
- Every action deserves feedback.
- Reduce unnecessary clicks.
- Progressive disclosure over complexity.
- Fast interfaces build confidence.
- Prevent errors whenever possible.
- Recovery should always be easy.
- Respect the user's time.
