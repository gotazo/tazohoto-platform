# Glossary

## Application

A complete product built using platform modules.

---

## Brand Token

A token representing the visual identity of a brand.

---

## Builder

A real estate developer or organization using the platform.

---

## Component

A reusable UI building block.

---

## Design Token

A reusable visual decision such as color, spacing, typography, or motion.

---

## Module

A self-contained business capability such as CRM, Projects, Inventory, or Sales.

---

## Package

A reusable code library within the monorepo.

---

## Platform

The shared foundation upon which all applications are built.

---

## Primitive Token

The raw foundation value used by higher-level tokens.

---

## Semantic Token

A token that describes purpose rather than appearance.

---

## Theme Profile

A complete visual profile that defines how semantic tokens are presented.

---

## Workspace

The pnpm monorepo containing all applications and packages.
